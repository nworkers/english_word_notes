# 02-multiline-word-meaning

## 목적

- 번호가 붙은 단어장 이미지에서 단어와 뜻이 여러 줄로 줄바꿈된 레이아웃을 검증합니다.

## 예시 형식

```text
1 sweep away
   ~을 휩쓸다, 쓸어가다, 휩쓸어 가다, 완전히 없애다

4 challenge
   n. 도전, 어려움
   v. (-하도록) 도전하다
```

## 확인 포인트

- 줄바꿈된 뜻 설명을 하나의 엔트리로 다시 묶을 수 있는지
- 번호를 `sourceNumber`로 보존할 수 있는지
- 여러 품사와 여러 의미를 한 항목 안에서 유지할 수 있는지
