# 작업 기록

- 일시: 2026-04-11
- 작업자: Codex
- 작업 내용: 이미지 업로드 후 실제 OCR을 통해 단어 목록을 추출하는 흐름을 구현함

## 이번 작업에서 변경한 내용

- `tesseract.js` 패키지를 추가함
- 서버 API가 목 데이터 대신 실제 OCR 로직을 호출하도록 변경함
- 업로드된 이미지에서 텍스트를 인식하고 단어-뜻 쌍을 파싱하는 로직을 추가함
- 파싱이 애매할 때 경고 메시지를 반환하도록 함
- OCR 원문 텍스트를 화면에서 확인할 수 있게 함

## 구현 방식

- `/api/extract`에서 업로드된 파일을 받음
- 서버에서 `tesseract.js`로 OCR 수행
- OCR 결과 텍스트를 줄 단위로 정리
- 구분 기호가 있는 한 줄 형식 또는 인접한 두 줄 형식을 기준으로 단어-뜻 쌍 추출
- 쌍을 찾지 못한 경우 영어 단어만 우선 추출하고 `뜻 확인 필요`로 표시

## 확인 결과

- TypeScript 검사 통과
- Next.js 프로덕션 빌드 통과
- 메인 페이지 `/`와 `/api/extract` 라우트 빌드 확인

## 현재 한계

- OCR 정확도는 입력 이미지 품질에 크게 영향받음
- 모든 배치 형식을 완벽히 파싱하지는 않음
- `tesseract.js`의 언어 데이터 준비 상황에 따라 첫 실행 시간이 길어질 수 있음

- 단어와 뜻이 여러 줄에 걸친 경우 파싱 규칙 보강
- OCR 오인식 수정을 위한 검수 UI 검토
