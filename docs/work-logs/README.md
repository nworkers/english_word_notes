# Work Logs

이 디렉토리는 작업 진행 기록을 저장하기 위한 공간입니다.

## 기록 원칙

- 작업 단위별로 MD 파일을 생성합니다.
- 날짜와 식별 번호를 파일명에 포함합니다.
- 결정 사항, 변경 이유, 다음 작업을 함께 남깁니다.

## 파일명 예시

```text
20260411-001-initialize-project-structure.md
20260411-002-write-initial-spec.md
```

## 권장 템플릿

```md
# 작업 기록

- 일시:
- 작업자:
- 작업 내용:
- 결정 사항:
- 다음 할 일:
```
